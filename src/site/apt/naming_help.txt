                                                                                            reports.

                                                                                            exit.                       
--out                         <rat:report out = 'text'/>    <out>                         Define the output file where  
                                                                                            to write a report to        
                                                                                            (default is System.out).

                                noDefaultLicenses =                                         configuration. By default   
                                'text'/>                                                    all approved default        
                                                                                            licenses are used

                                <licenses>text</licenses>                                   license definitions.  May   
                              </rat:report>                                                 be followed by multiple

                                scanHiddenDirectories =                                                                 
                                'text'/>                                                                                
--addLicense                  <rat:report addLicense =      <addLicense>                  Add the default license       
                                'text'/>                                                    header to any file with an  
                                                                                            unknown license that is     
                                                                                            not in the exclusion list.  
                                                                                            By default new files will   
                                                                                            be created with the         
                                                                                            license header, to force    
                                                                                            the modification of         
                                                                                            existing files use the      
                                                                                            --force option.             
--force                       <rat:report force = 'text'/>  <force>                       Forces any changes in files   
                                                                                            to be written directly to   
                                                                                            the source files (i.e. new  
                                                                                            files are not created).     
                                                                                            Only valid with             
                                                                                            --addLicense

                                'text'/>                                                    in the license headers,     
                                                                                            usually in the form of      
                                                                                            "Copyright 2008 Foo".       
                                                                                            Only valid with             
                                                                                            --addLicense                
--exclude                     <rat:report>                  <exclude>                     Excludes files matching       
                                <exclude>text</exclude>                                     wildcard <Expression>. May  
                              </rat:report>                                                 be followed by multiple     
                                                                                            arguments. Note that '--'   
                                                                                            or a following option is    
                                                                                            required when using this    
                                                                                            parameter.                  
--exclude-file                <rat:report excludeFile =     <excludeFile>                 Excludes files matching       
                                'text'/>                                                    regular expression in the   
                                                                                            input file.

--xml                         <rat:report xml = 'text'/>    <xml>                         Output the report in raw XML  
                                                                                            format.  Not compatible     
                                                                                            with -s
